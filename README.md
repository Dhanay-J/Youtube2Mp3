# Youtube2Mp3
Simple GUI to Convert Youtube Video to MP3 Using pytube and ttkbootstrap

`Python 3.12.0`

- Installation
    - `git clone https://github.com/Dhanay-J/Youtube2Mp3.git`
    - `cd Youtube2Mp3`
    - `python -mm venv .`
    - `.\Scripts\activate`
    - `pip install -r requirements.txt`
    - `python .\main.py`
